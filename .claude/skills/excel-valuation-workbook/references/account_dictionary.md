# 표준 계정 사전 (W2 계정 연속성 추적 근거)

`fs_clean.py`의 연도간 재분류 추적(금액 보존 매칭)을 **보강**하는 계정 상하위·흡수 관계 사전.
fs_clean은 1차로 금액 보존 매칭(결정론)을 하고, 애매하면 이 사전의 관계로 후보를 좁힌다.
**계정 "채택"은 평가인 판단** — 이 사전은 후보 근거일 뿐.

> ⚠️ W2(연도간 계정 연속성)와 W3(평가목적 재분류 WC/FA/NOA/IBD/OAL/EQU)는 **다른 층위**다.
> 이 사전은 W2용(공시 계정 명칭·이관). 평가유형 taxonomy는 `계정분류_모델아키텍처.md`(W3).

---

## 1. 자주 재분류되는 계정 (연도간 이관 후보)

회계처리방법·원칙 변경 시 아래 계정들이 서로 이관·병합·분할되는 경우가 많다. `fs_clean.py`가
당기/전기 교차 불일치를 잡으면 이 관계로 "어디로 갔나"를 좁힌다.

| 원 계정 | 흔한 이관처 | 사유 |
|---------|-------------|------|
| 기타유동자산 | 단기금융상품 / 미수금 / 선급금 / 선급비용 | 세분화(공시 정책 변경) |
| 단기금융상품 | 현금및현금성자산 ↔ 기타유동자산 | 만기 3개월 기준 재판단 |
| 매출채권 | 매출채권및기타채권 / 미수금 | 표시 통합·분리 |
| 기타비유동자산 | 장기금융상품 / 보증금 / 이연법인세자산 | 세분화 |
| 유형자산 | 사용권자산 | K-IFRS 1116 리스 도입(신설) |
| 기타유동부채 | 미지급금 / 미지급비용 / 예수금 / 계약부채 | 세분화 |
| 선수금 | 계약부채 | K-IFRS 1115 수익기준 변경 |
| 매입채무 | 매입채무및기타채무 | 표시 통합 |
| 장기차입금 | 유동성장기부채(1년내 만기 대체) | 만기 도래 재분류 |
| 이연법인세 | 자산/부채 상계 표시 변경 | 순액 표시 정책 |

## 2. 소계·총계 계정 (추적 제외)

아래는 구성항목의 합이므로 재분류 추적에서 제외(`fs_clean.py`가 "총계/합계/총액" 키워드로 자동 제외):

- 자산총계·부채총계·자본총계·부채와자본총계
- 유동자산·비유동자산·유동부채·비유동부채(소계)
- 매출총이익·영업이익·법인세차감전순이익·당기순이익(PL 단계 합계)

## 3. 명칭 변이 (동의어 — 같은 계정, 표기 차이)

| 표준 | 변이 |
|------|------|
| 매출액 | 수익, 영업수익, 매출 |
| 매출원가 | 영업비용(원가성) |
| 판매비와관리비 | 판관비, 판매관리비 |
| 현금및현금성자산 | 현금및현금등가물, 현금 |
| 이익잉여금 | 이익잉여금(결손금) |
| 종속기업투자 | 관계기업및종속기업투자주식 |

## 4. 사용 원칙

1. `fs_clean.py`는 **금액 보존 매칭이 1차**(사전 없이도 1:1·1:多 후보 생성).
2. 이 사전은 매칭이 애매할 때(여러 후보) **관계로 우선순위**를 줄 뿐, 자동 채택 근거가 아니다.
3. 다대다 재분류·명칭 신설은 **미해결로 표면화** → 평가인이 주석(회계정책 변경)과 대조해 확정.
4. 신규 이관 패턴을 발견하면 이 사전에 추가(누적 학습).
