# Val-Studio 브랜드 색상표 및 UI 컬러 시스템

글로벌 엔터프라이즈 SaaS의 표준 색상 시스템 설계 방식을 따르되, **'완벽한 톤앤매너의 일치(Tonal Cohesion)'**를 위해 앱 내의 모든 무채색과 상태 색상에 메인 브랜드 컬러인 `Burgundy Wine(#9f1239)`의 온도를 미세하게 섞어 커스텀 팔레트를 정의합니다.

> [!IMPORTANT]
> 순수한 검은색(#000000)과 흰색(#FFFFFF), 그리고 채도가 높은 기본 원색(초록, 노랑)의 남용을 금지합니다. 모든 색상은 버건디의 '고급스럽고 차분한(Muted & Warm)' 분위기 아래 하나로 묶여야 합니다.

---

## 1. 원시 팔레트 (Primitive Palette)

### 🍷 Primary Scale (Burgundy Wine)
Val-Studio의 정체성입니다.
*   `brand-50` (`#fdf8f9`): 극도로 연한 핑크빛 (활성화된 데이터 행)
*   `brand-600` (`#b91c47`): 버튼 Hover 상태
*   `brand-800` (`#9f1239`): **[Core Brand Color]** 메인 액션 버튼, 활성 탭
*   `brand-900` (`#730b27`): 다크모드 브랜드 배경

### 🌫️ Warm Neutral Scale (Burgundy-Tinted Gray)
단순한 차가운 회색(Slate)이나 순백/순흑이 아닌, **버건디의 붉은 톤이 1~3% 미세하게 섞인 '따뜻한 회색(Warm Gray / Wine Gray)'**을 베이스로 사용하여 화면 전체의 통일감을 극대화합니다.
*   `neutral-50` (`#fcfbfb`): 앱 전체 기본 배경 (순백색이 아닌 아주 미세한 웜톤)
*   `neutral-200` (`#e8e4e5`): 엑셀 형태의 빽빽한 테이블 1px 실선 테두리
*   `neutral-500` (`#7a7173`): 보조 텍스트, 힌트
*   `neutral-800` (`#362f31`): 주요 텍스트, 헤더 (완전한 검은색이 아닌 깊고 어두운 자줏빛 회색)
*   `neutral-900` (`#241f20`): 다크모드 베이스 배경

---

## 2. 상태 알림 컬러 (Cohesive System Colors)
원색(쨍한 초록, 노랑)의 남용을 막고, 버건디의 클래식한 분위기와 어우러지도록 **채도를 낮추고 묵직하게 가라앉힌(Muted & Earthy)** 커스텀 상태 색상표입니다.

*   🟢 **Success (Sage Green / 세이지 그린):** `#5b7c65`
    *   싸보이는 형광 초록색 대신, 차분하고 고급스러운 세이지(허브) 그린을 사용하여 버건디 톤을 해치지 않고 '통과/성공'을 우아하게 표현합니다.
*   🟡 **Warning (Antique Gold / 앤틱 골드):** `#c49b47`
    *   기본 노란색 대신, 오래된 놋쇠나 금빛에 가까운 앤틱 골드를 사용하여 '검토 필요/주의' 상태를 무겁게 잡아줍니다.
*   🔴 **Error (Burnt Rust / 번트 러스트):** `#cf3a36`
    *   브랜드 색상(자줏빛 도는 다크 레드)과 에러 색상을 구분하기 위해, 에러 색상은 '녹슨 철(Rust)' 색상에 가까운 주황빛 레드(다크 코랄)를 사용하여 시스템적 오류임을 명확히 합니다.

---

## 2b. 보간 토큰 (구현 등재, 2026-07-17 — 전부 웜 틴트 유지)
프론트 구현 중 필요해진 중간 단계 — 모두 R 채널이 최상위인 웜 그레이(쿨 그레이 금지 유지).
*   `panel` (`#fffdfd`): 패널 표면(캔버스 neutral-50 위, 순백 대체)
*   `neutral-100` (`#f6f3f4`): 테이블 헤더 배경 · `#f9f7f8`: finding 배경
*   `neutral-300` (`#c9c2c4`): disabled 텍스트(밝은 면) · `#a89fa1`: disabled 버튼
*   `neutral-400` (`#b5adaf`): 다크 LNB 기본 텍스트 · `#5c5456`: 다크 LNB disabled
*   다크 위 유틸: `rgba(255,255,255,.04)` hover · `rgba(159,18,57,.18)` 활성 틴트
*   (관례) 버건디/다크 배경 위 텍스트는 `#fff` 허용 — 배경용 순백은 금지 유지.

## 3. 시맨틱 맵핑 (실무 적용)
코딩 시 위의 커스텀 헥스(Hex) 값을 Tailwind 환경설정(`tailwind.config.js`)에 변수로 등록하여 다음과 같이 사용합니다.

*   **텍스트:** `text-neutral-800` (순흑색 금지)
*   **배경:** `bg-neutral-50` (순백색 금지)
*   **테두리:** `border-neutral-200`
*   **성공 뱃지:** `bg-[#5b7c65] text-white`
